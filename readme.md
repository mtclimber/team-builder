# Team Builder


## API Routes

**Teams**
```
GET /api/teams/
POST /api/teams/

GET /api/teams/:team_id
PUT /api/teams/:team_id
```

**Members**
```
GET /api/members/
POST /api/members/

GET /api/members/:member_id
PUT /api/members/:member_id
```

**Partners**
```
GET /api/partners/
POST /api/partners/

GET /api/partners/:partner_id
PUT /api/partners/:partner_id
```

**Authentication**
```
POST /api/auth/register
POST /api/auth/login
POST /api/logout
```
